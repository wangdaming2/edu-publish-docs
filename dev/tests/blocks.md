# 提示块与自定义容器

VitePress 内置了丰富的自定义容器，以下是它们的渲染效果。

::: info 信息 (Info)
这通常用来展示一条附加信息，背景比较柔和。
:::

::: tip 提示 (Tip)
这通常用来展示有用的建议或小贴士，主题色高亮。
:::

::: warning 警告 (Warning)
这用来提示用户需要注意的事项。
:::

::: danger 危险 (Danger)
用于提示严重错误或具有破坏性的操作。
:::

::: details 点击查看详细信息 (Details)
这是一个可折叠的区块，非常适合放置不重要但可能会需要的内容，比如**错误日志**或**长代码片段**。
:::
