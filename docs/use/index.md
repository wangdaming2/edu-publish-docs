# 管理面板

欢迎查阅 AstrBot 的管理面板使用指南。本章将详细介绍如何通过 WebUI 访问和管理系统。

## 管理面板

项目提供了一个直观的管理面板，用于管理插件、查看日志和进行可视化配置。

### 访问管理面板

当您成功部署项目后，可以通过以下 URL 访问管理面板：

- 本地部署: `http://localhost:6185`
- 云端部署: `您的自定义域名`

### 登录

默认用户名和密码是 `admin` 和 `password` (请在部署后及时修改)。

## 功能概览

- **插件管理**: 支持实时安装、卸载和重载插件。
- **可视化配置**: 丰富的配置项，支持多种适配器。
- **日志系统**: 全面的运行日志，方便故障排查。

::: warning
请务必在部署后修改默认密码，以保障系统安全。
:::
