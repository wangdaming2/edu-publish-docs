# 管理面板 (WebUI)

AstrBot 提供了一个现代化的 Web 管理界面，让您可以直观地配置机器人。

## 主要功能

- **仪表盘**: 实时查看消息统计、内存占用和运行状态。
- **适配器管理**: 轻松接入 QQ、微信、飞书等平台。
- **AI 提供商**: 配置 OpenAI、Anthropic、DeepSeek 等 AI 模型。
- **实时日志**: 无需进入终端，直接在网页端查看运行日志。

## 访问方式

默认访问地址为：`http://localhost:6185`

::: tip
如果您在云服务器部署，请确保防火墙已放行 `6185` 端口。
:::
