# SubAgent 编排

支持多代理（Multi-Agent）协同工作，让复杂的任务变得井井有条。

## 什么是 SubAgent

SubAgent 是指在主 AI 之下，专门负责某个子领域的 AI 代理，例如：
- **搜索代理**: 负责搜集资料。
- **写作代理**: 负责整理文档。
- **校对代理**: 负责检查错误。

## 编排流程

1. **新建编排**: 在 WebUI 的“编排管理”中新建。
2. **连接节点**: 像流程图一样连接不同的代理。
3. **测试运行**: 实时观察每个代理的工作进度。

::: tip
SubAgent 编排能够显著提高长文本生成和复杂逻辑推理的准确度。
:::
